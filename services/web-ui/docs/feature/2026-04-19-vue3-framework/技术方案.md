# Web: Vue 3 框架 - 技术方案

## 背景与目标

根据 MVP 需求文档，Web 模块作为 CLI 核心主体的扩展渲染层，需要通过 Vue 3 + Vite 构建现代化的前端框架。本阶段目标是搭建稳定可扩展的前端项目结构，为后续 Dashboard、MCP 配置、定时任务等功能的开发奠定基础。

**架构定位**：
- CLI 是核心主体，Web/Electron 是扩展渲染层
- Web 端定位：弥补 CLI 的可视化短板（Dashboard 图表、配置表单）
- Web 启动方式：通过 CLI 命令 `/help-web`、`/dashboard-web`、`/config-web` 等启动对应页面
- 非必须组件：CLI 独立运行，Web 仅作为可视化扩展

**设计规范**：Web UI 设计必须严格遵循 `DESIGN.md` 中的 Ollama 风格规范：
- 纯白背景 (`#ffffff`) + 灰度色系（无彩色）
- SF Pro Rounded 标题字体 + system-ui 正文字体
- 二元 border-radius 系统：12px（容器）| 9999px（交互元素）
- 零阴影，通过背景色和边框实现层次
- Pill-shaped 按钮和交互元素
- 字体权重：400（正文）| 500（标题）

## 需求概述

本任务需要完成 Web 前端项目框架的搭建，具体包括：

1. **项目初始化**：基于 Vite 创建 Vue 3 + TypeScript 项目
2. **设计系统集成**：将 DESIGN.md 规范转化为 TailwindCSS 配置和组件样式
3. **路由配置**：搭建 Vue Router 路由体系，配置 `/`（Help首页）、`/dashboard`、`/mcp`、`/config`、`/timer` 等路由
4. **HTTP 客户端**：封装 Axios 实例（baseURL: `http://127.0.0.1:8080`），统一处理 API 请求和响应拦截
5. **基础布局**：实现侧边栏导航组件，支持响应式折叠
6. **主题系统**：支持 light/dark/system 三种模式，本地存储偏好
7. **Help 首页**：作为应用首页，介绍项目功能、目标和项目介绍
8. **状态管理**：Pinia 状态管理基础配置

**CLI 命令规范**：Web 功能通过 CLI 命令 `/xx-web` 形式调用，如：
- `/help-web` → Help 首页
- `/dashboard-web` → Dashboard 页面
- `/mcp-web` → MCP 配置页面
- `/timer-web` → 定时任务页面
- `/config-web` → 系统配置页面

## 技术方案

### 方案选型

- **技术栈**：Vue 3.3+ / TypeScript 5.0+ / Vite 4.0+
- **路由**：Vue Router 4.0+（Hash 模式，避免服务端配置）
- **HTTP**：Axios 1.0+，baseURL: `http://127.0.0.1:8080`
- **样式**：TailwindCSS 3.0+（原子化 CSS，快速开发）
- **状态管理**：Pinia（Vue 3 官方推荐）
- **构建工具**：Vite 4.0+（ESM + HMR）

### 核心实现步骤

**Step 1: 初始化 Vite 项目结构**

1. 创建 `package.json`，配置依赖：
   - `vue@^3.4.0`
   - `vue-router@^4.3.0`
   - `pinia@^2.1.0`
   - `axios@^1.6.0`
   - `tailwindcss@^3.4.0`
   - `vite@^5.0.0`
   - `typescript@^5.3.0`
   - `@vitejs/plugin-vue@^5.0.0`
   - `vue-tsc@^1.8.0`

2. 创建 `vite.config.ts`：配置 Vue 插件、代理（API 请求代理到 `127.0.0.1:8080`）、别名

3. 创建 `tsconfig.json`：配置 TypeScript 编译选项，`strict: true`

4. 创建 `tailwind.config.js`：**扩展 DESIGN.md 规范到 TailwindCSS 配置**
   - 扩展灰度色板：`gray-50: #fafafa` 到 `gray-950: #030303`
   - 自定义字体堆叠：SF Pro Rounded / system-ui / ui-monospace
   - 自定义 border-radius：`rounded-container: 12px` / `rounded-pill: 9999px`

5. 创建 `postcss.config.js`：PostCSS 配置

6. 创建 `index.html`：Vue 挂载点，引入 TailwindCSS，设置基础字体

**Step 2: 配置 Vue Router**

1. 创建 `src/router/index.ts`
2. 配置路由表：
   - `/` -> HelpView（**作为默认首页，展示项目介绍和功能说明**）
   - `/dashboard` -> DashboardView（服务状态、统计图表）
   - `/mcp` -> MCPView（MCP 配置管理）
   - `/timer` -> TimerView（定时任务管理）
   - `/config` -> ConfigView（系统配置）
3. 使用 `createWebHashHistory()` 路由模式
4. 配置路由守卫（预留，后续鉴权扩展）

**Step 3: 封装 Axios 实例**

1. 创建 `src/api/client.ts`
2. 封装要点：
   - 创建 axios 实例，baseURL: `http://127.0.0.1:8080`
   - 请求拦截器：添加 `Content-Type: application/json`
   - 响应拦截器：统一错误处理（401、500、网络错误）
   - 超时配置：10 秒
3. **API 模块待开发时确定**：具体的 API 函数（`health.ts`、`knowledge.ts`、`timer.ts`、`mcp.ts`、`settings.ts`）将在实际开发时根据 server-go 实际路由定义进行创建

**Step 4: 创建基础布局组件**

1. 创建 `src/components/Layout/Sidebar.vue`
2. 实现固定左侧导航栏：
   - Logo + 服务名称 "Ddo"
   - Help（首页）、Dashboard、MCP、定时任务、配置 菜单项
   - 响应式折叠（移动端隐藏，菜单按钮触发）
3. 创建 `src/components/Layout/Layout.vue`：Sidebar + `<router-view />`

**Step 5: 实现主题系统**

1. 创建 `src/stores/theme.ts`（Pinia Store）
2. 支持三种主题模式：`light`、`dark`、`system`
3. 监听 `prefers-color-scheme` 实现自动适配
4. 主题偏好存储到 `localStorage`

**Step 6: 创建 Help 首页（项目介绍）**

1. 创建 `src/views/Help/HelpView.vue`
2. **内容结构**：
   - **项目目标**：Ddo 是一个本地化的 AI 开发助手平台，目标是提供可控的 AI 能力
   - **核心功能**：服务管理（MySQL/server-go/llm-py）、知识库管理、定时任务、MCP 工具扩展
   - **CLI 命令**：列出主要命令，格式为 `/xx-web`（如 `/help-web`、`/dashboard-web`、`/mcp-web`、`/timer-web`、`/config-web`）
   - **快捷入口**：跳转到 Dashboard、MCP 配置等页面
3. **设计风格**：遵循 DESIGN.md，简洁、纯白背景、灰度色系

**Step 7: 创建基础页面组件**

1. 创建各路由对应的空视图组件：
   - `src/views/Dashboard/DashboardView.vue`
   - `src/views/MCP/MCPView.vue`
   - `src/views/Timer/TimerView.vue`
   - `src/views/Config/ConfigView.vue`

2. 创建 `src/App.vue`：加载 Layout，设置主题类名

**Step 8: 创建全局样式和组件库**

1. 创建 `src/styles/main.css`
2. 定义 DESIGN.md 规范中的样式变量：
   ```css
   :root {
     --color-primary: #000000;
     --color-background: #ffffff;
     --color-surface: #fafafa;
     --color-border: #e5e5e5;
     --color-muted: #737373;
     --color-light: #a3a3a3;
     --radius-container: 12px;
     --radius-pill: 9999px;
   }
   ```

3. 创建基础组件（遵循 DESIGN.md 规范）：
   - `src/components/ui/Button.vue`：Gray Pill / White Pill / Black Pill 按钮
   - `src/components/ui/Card.vue`：12px 圆角容器
   - `src/components/ui/Input.vue`：Pill 形状输入框
