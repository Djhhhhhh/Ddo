## 功能验证清单

### 1. Electron 主进程

- [ ] `electron main.ts` 启动成功，无报错
- [ ] 主窗口正常创建（900x700）
- [ ] 加载 Web UI 地址正确
- [ ] 进程正确监听端口

### 2. 系统托盘

- [ ] 托盘图标显示在系统托盘区域
- [ ] 右键菜单显示 "显示窗口"、"设置"、"退出"
- [ ] 单击托盘图标切换主窗口显示/隐藏
- [ ] 有通知时托盘图标变化

### 3. 灵动岛悬浮窗口

- [ ] 窗口显示在屏幕右上角（320x120）
- [ ] 无标题栏、置顶、透明背景
- [ ] 显示通知标题和内容
- [ ] 5 秒后自动消失
- [ ] 操作按钮正常显示（完成、5分钟后提醒、查看详情）

### 4. 系统通知

- [ ] 普通通知显示系统通知中心
- [ ] 重要通知显示灵动岛
- [ ] 紧急通知同时显示灵动岛和系统通知
- [ ] 点击通知打开主窗口

### 5. IPC 通信

- [ ] 渲染进程接收 `notification:show` 事件
- [ ] 点击操作按钮发送 `notification:action` 到主进程
- [ ] 窗口控制 `window:toggle` 正常工作

### 6. 全局快捷键

- [ ] Ctrl+Shift+D 显示/隐藏主窗口

### 7. 后端连接

- [ ] WebSocket 连接 `ws://localhost:8080/ws/notify`
- [ ] 备用轮询 `GET /api/notifications/subscribe`
- [ ] 接收通知事件并正确展示

## 测试命令

```bash
# 编译 Vue Web UI
npm run build

# 编译并启动 Electron
npm run electron:dev

# 仅启动（需先编译）
node electron.mjs
```

## curl 测试示例

```bash
# 测试通知轮询接口
curl -X GET http://localhost:8080/api/notifications/subscribe

# 模拟推送通知（需后端支持）
curl -X POST http://localhost:8080/api/notifications \
  -H "Content-Type: application/json" \
  -d '{"title":"测试通知","body":"这是一条测试通知","level":"important"}'
```
