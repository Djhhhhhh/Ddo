## API 验证 (curl)

```bash
# 列出知识库
curl -X GET http://127.0.0.1:8080/api/v1/knowledge?page=1&page_size=20

# 搜索知识库
curl -X GET "http://127.0.0.1:8080/api/v1/knowledge/search?q=关键词"

# 获取知识详情
curl -X GET http://127.0.0.1:8080/api/v1/knowledge/{uuid}

# RAG 问答
curl -X POST http://127.0.0.1:8080/api/v1/knowledge/ask \
  -H "Content-Type: application/json" \
  -d '{"question":"如何配置 MCP？","top_k":3}'
```
