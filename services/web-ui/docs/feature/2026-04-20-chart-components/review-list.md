## 功能验证清单

### 图表组件

- [ ] **BarChart 柱状图** - Dashboard 中的 LLM 调用趋势
  - 预期结果: 显示近7天的调用量柱状图，灰度配色

- [ ] **PieChart 饼图** - Dashboard 中的定时任务状态分布
  - 预期结果: 显示运行中/已暂停的占比

- [ ] **图表数据更新** - 刷新数据后
  - 预期结果: 图表自动重新渲染

### Dashboard 自动刷新

- [ ] **30秒自动刷新** - 等待30秒
  - 预期结果: 数据自动更新，lastUpdateTime 变化

- [ ] **最后更新时间显示** - 刷新后
  - 预期结果: 显示正确的时间格式

- [ ] **手动刷新按钮** - 点击刷新按钮
  - 预期结果: 立即获取最新数据

- [ ] **刷新中状态** - 点击刷新按钮后
  - 预期结果: 按钮显示"加载中..."，禁用点击

### 错误处理

- [ ] **server-go 不可用** - 后端服务关闭时
  - 预期结果: 显示错误提示"获取指标数据失败"

- [ ] **数据为空** - 指标数据为空时
  - 预期结果: 图表显示"暂无数据"或0值
