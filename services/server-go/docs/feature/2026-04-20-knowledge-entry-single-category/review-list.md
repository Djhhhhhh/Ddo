- [ ] **AI 分析返回多个分类时，只取第一个** - 预期结果: 数据库只存储一个分类关联
- [ ] **用户未提供 category 时，使用 AI 分析的第一个分类** - 预期结果: category 字段正确填充
- [ ] **用户提供了 category 时，优先使用用户提供的值** - 预期结果: 以用户输入为准
- [ ] **Categories 数组只返回包含主分类的单一元素数组** - 预期结果: `categories` 字段如 `["技术"]`

### 分类自动创建

- [ ] **AI 分析出新分类时，自动创建分类** - 预期结果: 新分类出现在分类列表
- [ ] **AI 分析出已有分类时，直接使用已有分类 ID** - 预期结果: 不创建重复分类

## API 测试

### 创建词条（正常情况）
```bash
curl -X POST http://localhost:8080/api/knowledge/entries \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Go 语言最佳实践",
    "content": "Go 语言是一种高效的编程语言，适合构建微服务...",
    "source": "技术博客"
  }'
```
**预期响应**: 返回的 `categories` 只包含一个元素

### 创建词条（指定分类）
```bash
curl -X POST http://localhost:8080/api/knowledge/entries \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Docker 入门指南",
    "content": "Docker 是一个容器化平台...",
    "category": "运维",
    "source": "官方文档"
  }'
```
**预期响应**: `category` 为 "运维"，`categories` 为 `["运维"]`
